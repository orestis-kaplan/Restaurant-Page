# Restaurant-Page

Author: [Orestis Kaplanis](https://github.com/userman95)

## Installation Guide
### Requirements
```
node -v >= v8.16.0

npm -v >=  6.4.1
 
webpack -v >= 3.5.6

```
## Setup
#### Clone project
```
git clone git@github.com:userman95/Restaurant-Page.git
cd ./Restaurant-Page
```
#### Install npm 
```
npm install
```
#### Build project
```
webpack
```
Open up the `index.html` and you are ready to go
